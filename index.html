Surgimento da Internet

Voltar

A internet é um sistema global de redes de computadores interconectados que fornecem uma variedade de serviços e recursos de informação. Ela não surgiu de uma única vez, mas evoluiu ao longo do tempo com o desenvolvimento de tecnologias e padrões de comunicação.

Os primórdios da internet remontam à década de 1960, quando o Departamento de Defesa dos Estados Unidos iniciou pesquisas sobre redes de computadores interconectadas. O projeto ARPANET (Advanced Research Projects Agency Network) foi lançado em 1969 como uma rede experimental para compartilhamento de informações entre instituições acadêmicas e de pesquisa.

No início, a ARPANET se conectaria apenas a algumas universidades e centros de pesquisa. No entanto, à medida que a tecnologia evoluía, mais redes foram conectadas à ARPANET e a troca de informações entre elas se tornou possível. Essa rede inicial evoluiu para o que hoje mudou de internet.

A criação do TCP/IP (Transmission Control Protocol/Internet Protocol) nos anos 1970 foi um marco fundamental para o desenvolvimento da internet moderna. O TCP/IP é um conjunto de protocolos que permite a comunicação entre diferentes redes de computadores e é a base da internet como a usamos hoje.

Ao longo das décadas seguintes, a internet cresceu rapidamente, com a adição de novas redes, instituições e usuários. A World Wide Web, desenvolvida por Tim Berners-Lee em 1989, trouxe uma forma mais fácil e intuitiva de navegar e acessar informações na internet, através de hipertextos e hiperlinks.

Nos anos 1990, a internet comercial se popularizou, e serviços como provedores de acesso à internet, correio eletrônico, salas de bate-papo e navegadores da web tornaram-se amplamente disponíveis para o público em geral. Isso impulsionou ainda mais o crescimento e a adoção em massa da internet em todo o mundo.

Desde então, a internet continua a se expandir e se transformar em uma plataforma poderosa para comunicação, comércio, entretenimento, educação e muito mais. Novas tecnologias, como a banda larga, conexões móveis, redes sociais e aplicativos baseados na web, têm moldado o cenário digital e ampliado as possibilidades oferecidas pela internet.

O desenvolvimento e a evolução da internet revolucionaram a forma como as pessoas se comunicam, obtendo informações e interagem entre si, tornando-se uma ferramenta essencial na sociedade contemporânea.

HTML

Voltar

Seu nome tem origem do inglês Hyper Text Markup Language, que significa “linguagem de marcação de hipertexto”.

Como assim “linguagem de marcação”? Pois é, o HTML não é considerado uma linguagem de programação como as demais e você já vai descobrir o motivo disso.

Imagine uma página web, como essa mesmo que você esta lendo agora. É possível perceber que existem diversos elementos separados, como cabeçalho, título, parágrafos, imagens e muitos outros. Toda a organização desses elementos é feita pelo HTML. Ele é utilizado para criar toda a estrutura da página e, para isso, utiliza as famosas tags (etiquetas) para sinalizar onde cada tipo de elemento será implementado.

Tags h1 até h6

Voltar

As Heading Tags (H1, H2, H3, ...) são recursos de programação HTML utilizados para destacar títulos e subtítulos de uma página. Elas são utilizadas na hierarquização e estrutura de uma página.

Elas facilitam a leitura, entendimento e até mesmo a indexação do conteúdo e sua utilização é bem mais simples do que parece.

Tag Mark

Voltar

representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcada como referência, devido à sua conversão em um contexto particular. Por exemplo, pode ser usado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque.

Tag Small

Voltar

Tag define um texto menor (como direitos autorais e outros comentários laterais).

Tag i

Voltar

representa uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Normalmente, é apresentado com o uso do tipo "itálico".

Tag U

Voltar

representa algum texto não articulado e com estilo diferente do texto normal, como palavras com erros ortográficos ou nomes próprios em texto chinês. O conteúdo interno geralmente é exibido com um sublinhado.

Tag Strong

Voltar

O elemento HTML indica que seu conteúdo tem forte importância, seriedade ou urgência. Os navegadores normalmente renderizam o conteúdo em negrito.

Tag OL

Voltar

O elemento HTML representa uma lista ordenada de itens — normalmente renderizada como uma lista numerada.

Tag UL

Voltar

representa uma lista não ordenada de itens, normalmente renderizada como uma lista com marcadores.

Tag li

Voltar

é usado para representar um item em uma lista. Deve estar contido em um elemento pai: uma lista ordenada ( ), uma lista não ordenada ( ) ou um menu ( ). Em menus e listas não ordenadas, os itens da lista geralmente são exibidos usando marcadores. Em listas ordenadas, eles geralmente são exibidos com um contador crescente à esquerda, como um número ou uma letra.

Tag a

Voltar

O elemento HTML (ou elemento âncora ), com seu atributo , cria um hiperlink para páginas da Web, arquivos, endereços de e-mail, locais na mesma página ou qualquer outra coisa que um URL possa endereçar.

Tag hr

Voltar

representa uma quebra temática entre elementos de nível de parágrafo: por exemplo, uma mudança de cena em uma história ou uma mudança de tópico dentro de uma seção.

Tag sub

Voltar

O elemento HTML especifica o texto embutido que deve ser exibido como subscrito apenas por motivos tipográficos. Subscritos são normalmente renderizados com uma linha de base reduzida usando texto menor.

Tag sup

Voltar

O elemento HTML especifica o texto embutido que deve ser exibido como sobrescrito apenas por motivos tipográficos. Os sobrescritos geralmente são renderizados com uma linha de base elevada usando um texto menor.

Tag blockquote

Voltar

tag especifica uma seção citada de outra fonte.

Tag del

Voltar

representa uma parte do texto que foi excluída de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

Tag P

Voltar

O elemento HTML

representa um parágrafo. Em HTML, parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário.

Tag abbr

Voltar
